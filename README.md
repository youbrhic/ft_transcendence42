# FT_Trancendation

FT_Trancendation is the **final project of the 42 Common Core**, where we built a full-stack web application that allows users to **play games remotely, interact socially, and manage their profiles**.  

The platform offers:  
- **Online Games:** Play **Ping Pong** and **Tic Tac Toe** with other users in real time.  
- **User Management:** Profiles, friendship settings, and secure authentication.  
- **Real-Time Chat:** Communicate with friends using **Socket.IO** for instant messaging.  

FT_Trancendation combines **entertaining gameplay with social interaction**, showcasing a complete full-stack web application built from scratch. It demonstrates both **frontend and backend skills**, real-time communication, and secure user management in a single project.
